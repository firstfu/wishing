## 專案簡介

許願池是一個讓用戶可以發布需求並與潛在滿足者建立聯繫的平台。核心功能包括：

1. 用戶可以支付 1 元台幣發布許願需求
2. 置頂功能需支付 5 元台幣
3. 其他用戶可瀏覽需求並聯繫發布者

該平台旨在建立一個簡單、高效的需求對接生態系統，幫助人們快速找到所需物品或服務的提供者。
